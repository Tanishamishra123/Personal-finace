## ✨ Key Features

- 🔐 **User Authentication** - Secure login/register with JWT & bcrypt
- 📊 **Real-Time Dashboard** - Interactive charts with Chart.js
- 💵 **Income Tracking** - Multiple income sources with categories
- 💳 **Expense Management** - Categorized spending with budget limits
- 📈 **Investment Portfolio** - Track stocks, crypto, and assets
- 🎯 **Savings Goals** - Set targets and monitor progress
- 💳 **Credit Card Manager** - Track balances and payments
- 📱 **Fully Responsive** - Works on mobile, tablet, and desktop
- 🌐 **REST API** - Complete backend with Express.js
- 🗄️ **MongoDB Database** - Persistent data storage

## 🚀 Quick Start

### Prerequisites
- Node.js v16+
- MongoDB v6+

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/YOUR_USERNAME/personal-finance-management.git
cd personal-finance-management

# 2. Install dependencies
cd backend
npm install

# 3. Start MongoDB (in separate terminal)
mongod

# 4. Seed database with demo data
node seed.js

# 5. Start server
npm run dev

# 6. Open in browser
# Open frontend/index.html
# OR visit http://localhost:5000
